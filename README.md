# Ozobot Fractions Lab

## Run locally

From this folder, start a local web server:

```bash
python3 -m http.server 8000
```

Then open:

- http://localhost:8000

## Install as an app

Once it is served over `http://localhost` or HTTPS, the site can be installed as a PWA:

- In Safari on iPad: use Share -> Add to Home Screen
- In Chrome/Edge: use the Install App prompt in the address bar

## Stop server

Press `Ctrl + C` in the terminal where the server is running.
